What it checks

Rule-based finance QA checks designed for spreadsheet review before management reporting.

Formula inconsistencies

Find formulas that break the pattern used by neighboring period cells.

Hardcoded values among formulas

Spot numeric overrides in rows that appear to be formula-driven.

Blank cells inside financial rows

Catch empty cells sitting between populated period values.

Large period-over-period variances

Flag unusual adjacent-period movements that may need explanation.

Total/subtotal formula mismatches

Review totals that may omit expected detail rows.
