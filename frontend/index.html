Security & privacy

  • Finance reports can contain sensitive business data.
  • The current MVP can be run locally, so files stay on your machine during local testing.
  • For a hosted beta, uploaded files should be processed only for analysis and not stored permanently.
  • Private or self-hosted deployment can be considered for stricter requirements.
